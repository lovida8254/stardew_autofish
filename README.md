# 🎣 Stardew Valley 자동 낚시 봇 (완전 자동)

화면 인식 + PD 제어로 낚시를 자동화합니다.
**완전자동** 모드를 켜면 캐스팅 → 입질 감지 → 후킹 → 미니게임까지 봇이 알아서 반복합니다.
**완전자동 OFF** 이면 캐스팅·후킹은 직접 하고 미니게임만 봇이 잡는 보조 모드로 동작합니다.

## 설치 (Windows)

```bash
pip install -r requirements.txt
```

## 기본 좌표

기본 `config.json`은 **1920×1080 창모드** 기준으로 트랙/입질 영역 좌표가 맞춰져 있습니다.
게임 창 위치·크기가 다르면 아래 캘리브레이션으로 다시 잡으세요.

## 🖥️ GUI 버전 (권장)

```bash
python gui.py
```

| 기능 | 설명 |
|---|---|
| ▶ 시작 / ■ 정지 | 봇 on/off. 게임 중 전역 핫키 **F8** |
| 완전자동 (F9) | 체크 시 캐스팅+입질+미니게임 전자동, 해제 시 미니게임만 보조 |
| 🖱 클릭 테스트 | 게임을 포커스하고 1.2초 클릭 유지 → **미니게임 바가 올라가면 입력 정상** |
| 입질 감지: 소리/화면 | **소리(권장)**: 입질음을 시스템 오디오로 감지 / 화면: '!' 변화 감지 |
| 🎯 트랙 영역 | 3초 후 전체화면 오버레이 → 초록 바가 오르내리는 세로 트랙 드래그 (ESC 취소) |
| ❗ 입질 영역 | (화면 감지용) 찌 위쪽 물 영역 드래그 |
| 실시간 미리보기 | 초록 바(초록 박스) / 물고기(빨간 원) 인식 상태 확인 |
| 파라미터 슬라이더 | kp·kd·데드존·fps + 캐스팅 힘·**입질 소리 배수/바닥값**·대기 한도 |
| 고급 | 초록 바 HSV, 물고기 임계(S하한/V상한) 직접 수정 |
| 💾 저장 | 조정값을 config.json에 저장 |

### 입질 소리 감지 (기본, 권장)
입질 시 나는 소리를 **스피커 출력(WASAPI 루프백)**으로 잡아서 후킹합니다.
- 게임 소리가 스피커/헤드셋으로 나오고 있어야 합니다(음소거 X).
- 미리보기 옆 **음량 / 발동** 값을 보세요. 조용할 땐 음량이 발동값보다 낮고,
  입질음이 나면 음량이 확 튀어 발동값을 넘습니다. 잘 안 잡히면 **입질 소리 배수**를
  낮추고, 다른 소리에 오작동하면 **바닥값/배수**를 올리세요.

### 완전자동 사용 순서

1. 낚싯대를 든 상태로 게임 창을 **포커스**하고, 마우스 커서를 던질 물 위에 둡니다.
2. GUI에서 **완전자동** 체크 → **시작(F8)**.
3. 봇이 자동으로 던지고, 입질 시 후킹하고, 미니게임을 잡고, 보상창을 닫고 다시 던집니다.

> 입질이 잘 안 잡히면 미리보기 아래 **입질 변화량** 값을 보며 "입질 민감도"를 조정하세요.
> (물결·찌 움직임보다 '!' 말풍선이 뜰 때 값이 확 커집니다. 그 사이 값으로 기준을 맞추면 됩니다.)

## CLI 버전

```bash
python main.py calibrate         # 미니게임 트랙 영역 선택
python main.py calibrate-bite    # 입질 감지영역 선택 (완전자동용)
python main.py tune              # 초록바/물고기 마스크 실시간 확인
python main.py run               # 실행
```

실행 중 핫키:
- **F8** : 일시정지 / 재개
- **F9** : 완전자동 ON/OFF
- **ESC** : 종료

## 동작 원리

1. `mss`로 트랙 영역만 초당 45회 캡처
2. OpenCV로 초록 바(HSV) + 물고기(초록 바 아님 & 배경 물보다 어두운 덩어리, **색 무관**) 위치 추출
3. **예측 제어**: 바가 물고기보다 아래면 눌러서 올리되, 바의 상승 속도를 재서 `lookahead`초 뒤 예상 위치가 물고기에 닿을 것 같으면 미리 뗀다 → 관성으로 지나쳐 위로 튀는 것을 방지하고 물고기를 바 안에 유지한다.
4. 완전자동 상태기계: `캐스팅 → 입질대기(변화량 감지) → 후킹 → 미니게임 → 보상` 반복
5. `pydirectinput`으로 마우스 입력 전송

## 튜닝 팁 (config.json)

| 항목 | 설명 |
|---|---|
| `lookahead` | 예측 시간(초, 기본 0.22). **위로 지나쳐 튀면 ↑**(더 일찍 뗌), 물고기를 **못 따라가면 ↓** |
| `aim_offset` | 조준 보정(px). +면 물고기를 바 중앙보다 살짝 아래로 조준 |
| `fps` | 처리 속도. 높을수록 속도 추정이 정확 (기본 60) |
| `fish_s_min` / `fish_v_max` | 물고기가 안 잡히면 `fish_v_max` ↑, 배경까지 잡히면 ↓ |
| `bar_min_area` | 해초 등을 바로 오인식하면 ↑ |
| `cast_power` | 캐스팅 힘(초). 길수록 멀리 |
| `cast_settle` | 캐스팅 후 입질 감시 시작까지 대기(초) |
| `bite_change` | 입질 민감도. 낮으면 물결에도 반응, 높으면 놓침 |
| `wait_timeout` | 입질 없을 때 재캐스팅까지(초) |
| `cast_target` | 캐스팅 전 커서를 옮길 물 위 지점(x,y, 1920×1080 기준) |

## ❗ 클릭이 게임에 안 먹힐 때 (가장 흔한 문제)

스타듀 낚시 바는 **게임 창이 포그라운드(최상위)일 때만** 좌클릭 유지에 반응합니다.
봇이 GUI 창에 포커스가 있는 상태로 클릭하면 클릭이 게임에 전달되지 않습니다.

- 봇에는 **게임 창 자동 포커스** 기능이 있어(`auto_focus: true`), 입력 직전 자동으로
  Stardew 창을 최상위로 올립니다. 창 제목이 다르면 `config.json`의 `game_title`을 맞추세요.
- 먼저 **🖱 클릭 테스트**로 확인하세요. 바가 올라가면 입력 정상입니다.
- 그래도 안 되면 **게임을 관리자 권한으로 실행 중**일 가능성이 큽니다. 이 경우
  봇도 **관리자 권한으로 실행**해야 클릭이 전달됩니다(Windows UIPI 제한).
